# 📢 Shopify Announcement Manager App

A full-stack **Shopify Embedded App** built with the **MERN Stack** that allows merchants to create announcements from the Shopify Admin dashboard, stores them in **MongoDB** for audit history, syncs them to **Shopify Shop Metafields** using the **GraphQL Admin API**, and displays them on the storefront through a **Theme App Extension**.

---

## ✨ Features

- 📋 Embedded Shopify Admin App
- ✍️ Create announcements from the admin dashboard
- 💾 Store announcements in MongoDB with timestamps
- 🔄 Sync announcements to Shopify Shop Metafields
- 🎨 Display announcements on the storefront using Theme App Extension
- 📜 Maintain announcement history
- ⚡ Built using Shopify GraphQL Admin API
- 🔐 Secure Shopify authentication

# 🚀 Installation

## 1. Clone the repository

```bash
git clone https://github.com/Aqib1239/shopify-announcement.git

cd shopify-announcement
```

---

## 2. Install dependencies

### Shopify App

```bash
npm install
```

### Backend

```bash
cd backend

npm install
```

---

# 🔑 Environment Variables

Create a `.env` file inside the **backend** folder.

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string
```

For the Shopify app, configure the required environment variables generated by Shopify CLI.

---

# 🗄️ Run MongoDB

If using MongoDB Atlas:

Update your connection string inside

```
backend/.env
```

If using Local MongoDB:

```
mongodb://127.0.0.1:27017/shopify-announcement
```

---

# ▶️ Run the Backend

```bash
cd backend

npm run dev
```

Server runs at:

```
http://localhost:5000
```

---

# ▶️ Run Shopify App

From the project root:

```bash
shopify app dev
```

The Shopify CLI will:

- Start the embedded app
- Generate Cloudflare tunnel
- Open Shopify Admin
- Launch GraphiQL
- Start Theme Extension preview

# 🧪 How It Works

## Step 1

Merchant enters an announcement.

Example

```
Sale 50% Off
```

---

## Step 2

Click

```
Save Announcement
```

---

## Step 3

Backend saves the announcement into MongoDB.

Example document

```json
{
  "_id": "...",
  "text": "Sale 50% Off",
  "createdAt": "...",
  "updatedAt": "..."
}
```

---

## Step 4

Backend sends GraphQL mutation to Shopify.

Metafield

```
Namespace

my_app

Key

announcement
```

---

## Step 5

Theme App Extension reads

```liquid
shop.metafields.my_app.announcement
```

---

## Step 6

Announcement appears on every storefront page.

```
📢 Sale 50% Off
```
